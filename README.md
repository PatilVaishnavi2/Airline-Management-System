# ✈️ Airline Management System

A Java-based Airline Management System designed to streamline airline operations such as flight scheduling, ticket booking, passenger management, and more. This project is ideal for learning object-oriented programming, GUI development, and basic database integration in Java.

## 📌 Features

- 🛫 Flight scheduling and management
- 🎫 Ticket booking and cancellation
- 👤 Passenger information management
- 🧑‍✈️ Admin and user login system
- 📊 Real-time flight status updates
- 🗃️ Data persistence using file handling or database (e.g., MySQL)

## 🛠️ Technologies Used

- Java (Core + Swing for GUI)
- JDBC (for database connectivity)
- MySQL (or file-based storage)
- NetBeans / Eclipse IDE (recommended)
